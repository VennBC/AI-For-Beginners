# 使用 RNN 进行词级文本生成

来自 [AI 初学者课程](https://github.com/microsoft/ai-for-beginners) 的实验作业。

## 任务

在这个实验中，您需要取任何书籍，并将其用作数据集来训练词级文本生成器。

## 数据集

欢迎您使用任何书籍。您可以在 [Project Gutenberg](https://www.gutenberg.org/) 找到很多免费文本，例如，这里是 [Alice's Adventures in Wonderland](https://www.gutenberg.org/files/11/11-0.txt)) 的直接链接，由 Lewis Carroll 编写。
