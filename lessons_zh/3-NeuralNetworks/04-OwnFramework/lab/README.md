# 使用我们自己的框架进行 MNIST 分类

来自 [AI 初学者课程](https://github.com/microsoft/ai-for-beginners) 的实验作业。

## 任务

使用 1 层、2 层和 3 层感知机解决 MNIST 手写数字分类问题。使用我们在课程中开发的神经网络框架。

## 起始笔记本

通过打开 [MyFW_MNIST.ipynb](MyFW_MNIST.ipynb) 开始实验

## 问题

作为本实验的结果，尝试回答以下问题：

- 层间激活函数是否影响网络性能？
- 我们是否需要 2 层或 3 层网络来完成此任务？
- 您在训练网络时是否遇到任何问题？特别是随着层数的增加。
- 网络的权重在训练期间如何表现？您可以绘制权重与轮次的最大绝对值来理解关系。
