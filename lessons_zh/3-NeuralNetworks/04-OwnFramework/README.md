## 机器学习的形式化

让我们从形式化机器学习问题开始。假设我们有一个带标签 **Y** 的训练数据集 **X**，我们需要构建一个模型 *f*，它将做出最准确的预测。预测的质量由**损失函数** &lagran; 衡量。经常使用以下损失函数：

* 对于回归问题，当我们需要预测一个数字时，我们可以使用**绝对误差** &sum;<sub>i</sub>|f(x<sup>(i)</sup>)-y<sup>(i)</sup>|，或**平方误差** &sum;<sub>i</sub>(f(x<sup>(i)</sup>)-y<sup>(i)</sup>)<sup>2</sup>
* 对于分类，我们使用**0-1 损失**（本质上与模型的**准确率**相同），或**逻辑损失**。

对于单层感知机，函数 *f* 被定义为线性函数 *f(x)=wx+b*（这里 *w* 是权重矩阵，*x* 是输入特征向量，*b* 是偏置向量）。对于不同的神经网络架构，此函数可以采用更复杂的形式。

> 在分类的情况下，通常希望获得相应类别的概率作为网络输出。为了将任意数字转换为概率（例如，归一化输出），我们经常使用 **softmax** 函数 &sigma;，函数 *f* 变为 *f(x)=&sigma;(wx+b)*

在上面的 *f* 定义中，*w* 和 *b* 称为**参数** &theta;=⟨*w,b*⟩。给定数据集 ⟨**X**,**Y**⟩，我们可以计算整个数据集上的总体误差作为参数 &theta; 的函数。

> ✅ **神经网络训练的目标是通过改变参数 &theta; 来最小化误差**

## 梯度下降优化

有一个众所周知的函数优化方法称为**梯度下降**。这个想法是我们可以计算损失函数关于参数的导数（在多维情况下称为**梯度**），并以使误差减少的方式改变参数。这可以形式化为：

* 通过某些随机值初始化参数 w<sup>(0)</sup>, b<sup>(0)</sup>
* 多次重复以下步骤：
    - w<sup>(i+1)</sup> = w<sup>(i)</sup>-&eta;&part;&lagran;/&part;w
    - b<sup>(i+1)</sup> = b<sup>(i)</sup>-&eta;&part;&lagran;/&part;b

在训练期间，优化步骤应该考虑整个数据集来计算（记住损失是通过所有训练样本的总和计算的）。然而，在现实生活中，我们取称为**小批量**的数据集的小部分，并基于数据子集计算梯度。因为每次随机取子集，这种方法称为**随机梯度下降**（SGD）。

## 多层感知机和反向传播

单层网络，正如我们在上面看到的，能够分类线性可分的类别。为了构建更丰富的模型，我们可以组合网络的几层。在数学上，这意味着函数 *f* 将具有更复杂的形式，并将在几个步骤中计算：
* z<sub>1</sub>=w<sub>1</sub>x+b<sub>1</sub>
* z<sub>2</sub>=w<sub>2</sub>&alpha;(z<sub>1</sub>)+b<sub>2</sub>
* f = &sigma;(z<sub>2</sub>)

这里，&alpha; 是**非线性激活函数**，&sigma; 是 softmax 函数，参数 &theta;=<*w<sub>1</sub>,b<sub>1</sub>,w<sub>2</sub>,b<sub>2</sub>*>。

梯度下降算法将保持不变，但计算梯度会更困难。给定链式微分法则，我们可以计算导数：

* &part;&lagran;/&part;w<sub>2</sub> = (&part;&lagran;/&part;&sigma;)(&part;&sigma;/&part;z<sub>2</sub>)(&part;z<sub>2</sub>/&part;w<sub>2</sub>)
* &part;&lagran;/&part;w<sub>1</sub> = (&part;&lagran;/&part;&sigma;)(&part;&sigma;/&part;z<sub>2</sub>)(&part;z<sub>2</sub>/&part;&alpha;)(&part;&alpha;/&part;z<sub>1</sub>)(&part;z<sub>1</sub>/&part;w<sub>1</sub>)

> ✅ 链式微分法则用于计算损失函数关于参数的导数。

请注意，所有这些表达式的左侧部分都是相同的，因此我们可以从损失函数开始有效地计算导数，并通过计算图"向后"进行。因此，训练多层感知机的方法称为**反向传播**，或'backprop'。
