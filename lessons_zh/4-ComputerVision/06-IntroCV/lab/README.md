# 使用光流检测运动

来自 [AI 初学者课程](https://aka.ms/ai-beginners) 的实验作业。

## 任务

考虑[此视频](palm-movement.mp4)，其中一个人的手掌在稳定背景上左右/上下移动。

<img src="../images/palm-movement.png" width="30%" alt="手掌运动帧"/>

**您的目标**是能够使用光流来确定视频的哪些部分包含上/下/左/右运动。

**延伸目标**是使用肤色实际跟踪手掌/手指运动，如[这篇博客文章](https://dev.to/amarlearning/finger-detection-and-tracking-using-opencv-and-python-586m) 或[这里](http://www.benmeline.com/finger-tracking-with-opencv-and-python/) 中所述。

## 起始笔记本

通过打开 [MovementDetection.ipynb](MovementDetection.ipynb) 开始实验

## 要点

有时，相对复杂的任务，如运动检测或指尖检测，可以纯粹通过计算机视觉来解决。因此，了解像 OpenCV 这样的库可以做什么是非常有帮助的。
