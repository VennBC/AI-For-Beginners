# 遗传算法

## [课前测验](https://ff-quizzes.netlify.app/en/ai/quiz/41)

**遗传算法**（GA）基于 AI 的**进化方法**，其中使用群体进化的方法来获得给定问题的最优解。它们由 [John Henry Holland](https://wikipedia.org/wiki/John_Henry_Holland) 在 1975 年提出。

遗传算法基于以下思想：

* 问题的有效解决方案可以表示为**基因**
* **交叉**允许我们将两个解决方案组合在一起以获得新的有效解决方案
* **选择**用于使用某个**适应度函数**选择更优的解决方案
* **变异**被引入以破坏优化的稳定性并使我们摆脱局部最小值

如果您想实现遗传算法，您需要以下内容：

 * 找到使用**基因** g&in;&Gamma; 编码我们问题解决方案的方法
 * 在基因集 &Gamma; 上，我们需要定义**适应度函数** fit: &Gamma;&rightarrow;**R**。较小的函数值对应于更好的解决方案。
 * 定义**交叉**机制以将两个基因组合在一起以获得新的有效解决方案 crossover: &Gamma;<sup>2</sub>&rightarrow;&Gamma;。
 * 定义**变异**机制 mutate: &Gamma;&rightarrow;&Gamma;。

在许多情况下，交叉和变异是操作基因作为数字序列或位向量的相当简单的算法。

遗传算法的具体实现可能因情况而异，但总体结构如下：

1. 选择初始群体 G&subset;&Gamma;
2. 随机选择在此步骤将执行的操作之一：交叉或变异
3. **交叉**：
  * 随机选择两个基因 g<sub>1</sub>, g<sub>2</sub> &in; G
  * 计算交叉 g=crossover(g<sub>1</sub>,g<sub>2</sub>)
  * 如果 fit(g)<fit(g<sub>1</sub>) 或 fit(g)<fit(g<sub>2</sub>) - 用 g 替换群体中的相应基因。
4. **变异** - 选择随机基因 g&in;G 并用 mutate(g) 替换它
5. 从步骤 2 重复，直到我们获得足够小的 fit 值，或直到达到步骤数的限制。
