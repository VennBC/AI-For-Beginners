# 致教育工作者

您想在课堂上使用本课程吗？请随意使用！

实际上，您可以通过使用 GitHub Classroom 在 GitHub 本身中使用它。

要做到这一点，请 Fork 此仓库。您需要为每节课创建一个仓库，因此您需要将每个文件夹提取到单独的仓库中。这样，[GitHub Classroom](https://classroom.github.com/classrooms) 可以分别选择每节课。

这些[完整说明](https://github.blog/2020-03-18-set-up-your-digital-classroom-with-github-classroom/)将为您提供如何设置课堂的想法。

## 按原样使用仓库

如果您想按当前状态使用此仓库，而不使用 GitHub Classroom，也可以这样做。您需要与您的学生沟通，告诉他们一起学习哪一节课。

在在线格式（Zoom、Teams 或其他）中，您可以为测验组建分组讨论室，并指导学生帮助他们准备学习。然后在特定时间邀请学生参加测验，并将他们的答案作为"问题"提交。如果您希望学生公开协作，您也可以对作业做同样的事情。

如果您更喜欢更私密的格式，请要求您的学生将课程逐课 Fork 到他们自己的 GitHub 仓库作为私有仓库，并授予您访问权限。然后他们可以私下完成测验和作业，并通过您课堂仓库上的问题提交给您。

有很多方法可以在在线课堂格式中实现这一点。请告诉我们什么最适合您！
