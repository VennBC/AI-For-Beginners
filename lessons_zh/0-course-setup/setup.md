# 开始使用本课程

## 您是学生吗？

请从以下资源开始：

* [学生中心页面](https://docs.microsoft.com/learn/student-hub?WT.mc_id=academic-77998-cacaste) 在此页面上，您将找到初学者资源、学生包，甚至可以获得免费认证券的方法。这是一个值得收藏并经常查看的页面，因为我们至少每月更新一次内容。

**学生们**，有几种方式可以使用本课程。首先，您可以直接在 GitHub 上阅读文本并查看代码。如果您想运行任何笔记本中的代码 - [请阅读我们的说明](./how-to-run.md)，并在[这篇博客文章](https://soshnikov.com/education/how-to-execute-notebooks-from-github/)中找到更多建议。

> **注意**：[如何运行本课程中的代码的说明](./how-to-run.md)

## 自主学习

但是，如果您想将本课程作为自主学习项目，我们建议您将整个仓库 Fork 到您自己的 GitHub 账户，然后自己或与小组一起完成练习：

* 从课前测验开始。
* 阅读课程介绍文本。
* 如果课程有额外的笔记本，请通读并执行代码。如果同时提供了 TensorFlow 和 PyTorch 笔记本，您可以专注于其中一个 - 选择您喜欢的框架。
* 笔记本通常包含一些挑战，需要您稍微调整代码以进行实验。
* 参加课后测验。
* 如果模块附有实验 - 完成作业。
* 访问[讨论板](https://github.com/microsoft/AI-For-Beginners/discussions)进行"大声学习"。

> 为了进一步学习，我们建议遵循这些 [Microsoft Learn](https://docs.microsoft.com/en-us/users/dmitrysoshnikov-9132/collections/31zgizg2p418yo/?WT.mc_id=academic-77998-cacaste) 模块和学习路径。

## 教学法

在构建本课程时，我们选择了两个教学原则：确保它是**基于项目的实践**课程，并包含**频繁的测验**。

通过确保内容与项目对齐，使过程对学生更具吸引力，并增强概念的记忆。此外，课前的小测验可以让学生专注于学习主题，而课后第二次测验可以确保进一步记忆。本课程设计灵活有趣，可以整体或部分学习。项目从小开始，在 12 周周期结束时变得越来越复杂。

> **关于测验的说明**：所有测验都包含[在此应用中](https://red-field-0a6ddfd03.1.azurestaticapps.net/)，总共 50 个测验，每个测验包含三个问题。它们从课程中链接，但测验应用可以在本地运行；请按照 `etc/quiz-app` 文件夹中的说明操作。
